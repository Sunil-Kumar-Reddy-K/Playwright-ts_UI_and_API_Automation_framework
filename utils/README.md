# Utilities

Helper modules for Gmail API integration — used by the email automation tests.

## Files

| File | Purpose |
|---|---|
| `gmailService.ts` | Gmail API client with OAuth2 authentication |
| `emailHelpers.ts` | Email parsing and content extraction helpers |
| `generateRefreshToken.ts` | Script to generate OAuth2 refresh tokens |
| `testGmailConnection.ts` | Connection verification for Gmail API setup |
